Lista de etiquetas HTML

Definición: Las etiquetas HTML son fragmentos de texto rodeados por corchetes angulares < >, que tienen funciones y usos específicos y se utilizan para escribir código HTML.

En la siguiente tabla estan todas las etiquetas disponibles en HTML.

Etiqueta Descripción Atributos principales Ejemplos
!DOCTYPE tipo de documento (versión de html empleada)
html engloba todo el documento lang
<!-- ... --> comentario (sólo visible en el código fuente)
